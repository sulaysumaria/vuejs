# VueJS-CrashCourse

### Running locally

```bash
$ git clone
$ cd vuejs-crashcourse
$ npm i -g http-server
$ http-server -c-1 .
```

## Topics covered

- Data and Methods
- Data binding
- Events
- Event modifiers
- Keyboard events
- Two way data binding
- Computed properties
- Dynamic CSS Classes
- Conditionals
